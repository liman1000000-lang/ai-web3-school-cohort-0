## Repository Structure

```text
README.md                 # Project overview and links
profile.md                # Learner profile and preferences
learning-plan.md          # Lightweight learning plan
daily/                    # Daily learning notes and check-in drafts
tasks/                    # Task notes and proof-of-work records
experiments/              # Small experiments and content drafts
handbook-feedback/        # Feedback for Handbook pages
hackathon/                # Hackathon ideas, plans, and submissions
submissions/              # Submitted check-ins or task records
templates/daily-note.md   # Daily note template
templates/task-note.md    # Task note template
```

## Working Principle

This repo is a proof-of-work workspace: every note should help future review, public indexing, or Handbook feedback. Keep entries lightweight, clear, and safe to publish.
