# Learning Plan

## Current Strategy

Because the learner is new to AI, new to Web3, and currently no-code, the default learning path is:

1. Understand the day's course topic in plain Chinese.
2. Capture 3–5 key ideas in a daily note.
3. Turn one question or confusion into a Handbook feedback item when useful.
4. Draft a short public check-in.
5. Keep all public notes free of secrets and private information.

## Daily 30-Minute Template

- 5 min: Open WCB Learning and confirm today's course/task/check-in link.
- 10 min: Read the relevant Handbook section.
- 10 min: Write the daily note: what I learned, what confused me, one next action.
- 5 min: Generate or polish the check-in draft and submit manually on WCB/check-in platform.

## Path Levels

- Minimum path: one daily note + one check-in draft.
- Recommended path: daily note + one question/feedback item + check-in draft.
- Challenge path: turn one concept into a short public explainer or content outline.
